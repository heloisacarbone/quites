# Quites!
Aplicação para divisão de contas

## Para rodar
-> No terminal   
	<b>node app.js   </b>   
-> No navegador   
	<b>http://localhost:8080/html/index.html</b>   

## Descrição
Projeto dividido em tres partes:
backend - server part
frontend - interface do usuário
database - funcoes para banco de dados
